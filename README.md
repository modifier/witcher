# Карта мира Ведьмака

Тут живёт мир Ведьмака: карта, границы королевств, города, замки, всякое такое. А вообще это типа как руководство по тому, как делать карты вымышленных миров. Можно переключаться между коммитами и смотреть, как дела делаются.

## Чё делать-то?

Проект сделан на Svelte. Мне просто понравился Svelte. Проект запускается привычным образом, сначала надо установить зависимости:

```bash
npm install
```

Ну а потом можно запустить сам проект:

```bash
npm run dev
```

Сама карта будет по адресу [localhost:5000](http://localhost:5000).
